Simple Math (Addition and Subtraction)

Here are some basic math problems like 1+1, 1+2, 1-1, and more:

1 + 1 = 2

1 + 2 = 3

1 + 3 = 4

1 - 1 = 0

2 + 2 = 4

2 + 3 = 5

3 - 1 = 2

4 - 1 = 3

5 - 1 = 4

Simple Math Quiz:

What is 1 + 3?

Algebra

In algebra, we solve for unknowns using equations. For example:

If x + 2 = 5, then x = 3.

If 2x = 8, then x = 4.

If 3x + 2 = 11, then x = 3.

Algebra Quiz:

If 3x + 4 = 16, what is x?

Olympiad Math

Here are some harder math problems commonly found in math Olympiads:

If x² + 4x - 5 = 0, solve for x. (Answer: x = 1 or x = -5)

Find the sum of the first 100 natural numbers. (Answer: 5050)

Olympiad Math Quiz:

Solve for x: x² - 6x + 9 = 0
